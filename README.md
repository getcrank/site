# Crank Site

This is the source code for the [Crank](https://github.com/getcrank/crank) website. We use Vite + React as our static site generator.

## Getting Started

```
npm install
```

## Development

Run commands:

```
pnpm run dev         # run a localhost:5173 dev server with HMR
pnpm run build       # build site into dist/
pnpm run preview     # preview the production build locally
pnpm run lint        # run ESLint
```
